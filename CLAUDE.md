# Monorepo Setup

Follow these steps in order when scaffolding the repository from scratch:

1. **Initialize root package**

   ```
   pnpm init
   ```

   Set `"private": true` in the root `package.json`.

2. **Configure pnpm workspaces**
   Create `pnpm-workspace.yaml`:

   ```yaml
   packages:
     - 'packages/*'
     - 'apps/*'
   ```

3. **Initialize Nx**

   ```
   npx nx@latest init
   ```

   Sets up `nx.json` and integrates with pnpm workspaces.

4. **Scaffold portex package**

   ```
   pnpm create vite packages/portex --template vanilla-ts
   ```

   Then adjust `vite.config.ts` for library mode (ESM + CJS output with `vite-plugin-dts`).

5. **Add dev dependencies** (at root):

   ```
   pnpm add -Dw vitest playwright @playwright/test eslint prettier husky lint-staged @changesets/cli vite-plugin-dts typescript
   ```

6. **Configure tooling**
   - `tsconfig.json` ‚ÄĒ generated by scaffold, extend as needed
   - `vite.config.ts` ‚ÄĒ library mode with `vite-plugin-dts`
   - `vitest.config.ts` or inline in `vite.config.ts`
   - `.eslintrc` / `eslint.config.js`
   - `.prettierrc`
   - `npx husky init` + configure lint-staged in `package.json`
   - `npx changeset init`

7. **Scaffold playground app** (optional)
   ```
   pnpm create vite apps/playground --template vanilla-ts
   ```

# Pre-Publish Checklist

- [ ] Wire protocol uses `__portex` (not `__msglink`)
- [ ] `package.json`: `name`, `version`, `main`, `module`, `types`, `exports`, `files`, `sideEffects: false`
- [ ] Tree-shakeable exports
- [ ] Bundle size < 2KB gzipped
- [ ] Test coverage: rpc, abort, progress, events, worker, destroy, ready, transfer
- [ ] `ctx.progress()` is a no-op after call is settled or aborted
- [ ] Abort messages are gracefully ignored if call has already resolved
- [ ] `waitReady: true` correctly queues and flushes messages
- [ ] Transfer works on `.call()`, `.emit()`, `ctx.progress()`, and `ctx.transfer()`
- [ ] BroadcastChannel transport: transfer is ignored (BroadcastChannel limitation)
- [ ] README has a quick-start ‚Č§ 15 lines
